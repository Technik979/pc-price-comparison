# Техническое задание: PC Price Monitor

## Название проекта
**PC Price Monitor** — веб-приложение для мониторинга и сравнения цен на комплектующие для ПК в магазинах Citilink и DNS.

## Роли пользователей

| Роль | Возможности |
|------|------------|
| Гость | Просмотр каталога, поиск, сравнение цен, просмотр графиков |
| Авторизованный пользователь | Всё вышеперечисленное + добавление в список отслеживания, задание целевой цены |

## Описание моделей

### Category — категория товара
- `name` — название (Ноутбуки, SSD, ...)
- `slug` — URL-идентификатор
- `icon` — эмодзи-иконка

### Store — магазин
- `name` — название (Citilink / DNS)
- `url` — ссылка на сайт
- `color` — цвет бренда для UI

### Component — комплектующее
- `name` — наименование товара
- `category` — FK → Category
- `specs` — краткие характеристики (строка)
- `slug` — URL-идентификатор

### PriceHistory — история цен
- `component` — FK → Component
- `store` — FK → Store
- `price` — цена в рублях
- `scraped_at` — дата и время записи

### Watchlist — список отслеживания
- `user` — FK → User (Django auth)
- `component` — FK → Component
- `target_price` — целевая цена (опционально)
- `added_at` — дата добавления
- `is_notified` — флаг уведомления

## Пользовательские истории (User Stories)

1. Как гость, я хочу видеть каталог товаров с ценами из двух магазинов, чтобы быстро найти нужное комплектующее.
2. Как гость, я хочу искать товар по названию в режиме реального времени (AJAX), чтобы не перезагружать страницу.
3. Как гость, я хочу видеть интерактивный график динамики цен за 30 дней, чтобы понять, растёт или падает цена.
4. Как авторизованный пользователь, я хочу добавить товар в список отслеживания и задать целевую цену, чтобы не пропустить выгодное предложение.
5. Как авторизованный пользователь, я хочу видеть индикатор «цена достигнута», когда текущая цена упала ниже целевой.

## Технологический стек

| Слой | Технологии |
|------|-----------|
| Backend | Python 3.12, Django 4.x, Django REST Framework |
| Аналитика | Pandas (агрегация цен по категориям) |
| Визуализация | Plotly (интерактивные графики) |
| Frontend | Bootstrap 5.3, JavaScript (AJAX live search) |
| БД | SQLite (разработка), PostgreSQL (продакшн) |
| Деплой | PythonAnywhere |
