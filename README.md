# PC Price Monitor

Веб-приложение на **Django** для сравнения цен на электронику и бытовую технику в четырёх магазинах: **Citilink**, **DNS**, **М.Видео** и **Эльдорадо**.

## Возможности

### Каталог и поиск
- **252 товара** в 17 категориях с историей цен за 30 дней
- AJAX-поиск в реальном времени (debounce 280 мс) без перезагрузки страницы
- Фильтрация по категории, фильтры ноутбуков: диагональ, ОЗУ, процессор (AMD/Intel), тип SSD
- Сортировка по лучшей цене, подсветка самого выгодного магазина

### Цены и аналитика
- Цены из **4 магазинов** для каждого товара; прочерк, если товар отсутствует
- Интерактивный **Plotly-график** динамики цен за 30 дней (отдельная кривая на магазин)
- Аналитика по категориям — средняя / мин / макс цена через **Pandas**
- Топ сделок: товары с наибольшим разбросом цен между магазинами

### Добавление товаров
- Форма добавления товара — доступна без авторизации
- **Загрузка по ссылке**: вставьте URL из Citilink / DNS / М.Видео / Эльдорадо — название и цена заполнятся автоматически
- **Поиск во всех магазинах**: после загрузки одной ссылки система сама ищет тот же товар в остальных трёх магазинах (параллельный поиск — БД + live-scraping)
- Блок «похожие товары» прямо в форме добавления и на странице товара

### Профиль и отслеживание
- Регистрация / вход / выход
- **Список отслеживания** с целевой ценой и индикатором «цена достигнута»
- **Страница профиля**: статистика по вишлисту — количество товаров, достигнутые цели, суммарная экономия, сравнительный график цен по магазинам

### REST API
| Эндпоинт | Метод | Описание |
|---|---|---|
| `/api/prices/<pk>/` | GET | История цен товара |
| `/api/search/` | GET | Полнотекстовый поиск |
| `/api/parse-url/` | POST | Парсинг цены и названия по URL |
| `/api/find-prices/` | GET | Поиск цен товара во всех магазинах по названию |
| `/api/similar/` | GET | Похожие товары |

## Установка

```bash
git clone https://github.com/Technik979/pc-price-comparison.git
cd pc-price-comparison
pip install -r requirements.txt

cp .env.example .env          # заполните SECRET_KEY

python manage.py migrate
python manage.py update_prices --days 30   # заполнить БД демо-данными
python manage.py runserver
```

Откройте http://127.0.0.1:8000/

### Опции команды `update_prices`

```bash
python manage.py update_prices --days 60   # история за 60 дней
python manage.py update_prices --clear     # сбросить цены и заполнить заново
```
